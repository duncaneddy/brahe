# Orbital Dynamics Module

The orbital dynamics module provides functions for computing perturbation accelerations affecting satellite motion.

For conceptual explanations and usage examples, see [Orbital Dynamics](../../learn/orbital_dynamics/index.md) in the Learn section.

## Submodules

- [Gravity Models](gravity.md) - Point-mass and spherical harmonic gravity
- [Third-Body Perturbations](third_body.md) - Sun, Moon, and planetary effects
- [Atmospheric Drag](drag.md) - Drag force calculations
- [Atmospheric Density Models](atmospheric_density_models.md) - Density model implementations
- [Solar Radiation Pressure](solar_radiation_pressure.md) - SRP and eclipse modeling
- [Relativistic Effects](relativity.md) - General relativistic corrections
- [Ephemerides](ephemerides.md) - Celestial body position calculations
