Relative Coordinates¶

This module provides functions to convert between different representations of relative orbital coordinates.

brahe.relative_coordinates.rCARTtoRTN(x: NewType.<locals>.new_type) → numpy.ndarray¶

Compute the Earth-centered inertial to radial, along-track, cross-track (RTN) rotation matrix. Which, if applied to a position vector in the Cartesian frame, will transform that vector into the equivalent position vector in the RTN frame.

The RTN frame is also commonly refered to as the local-vertical, local-horizontal (LVLH) frame.

Parameters

  • x (np.ndarray) – Inertial state (position and velocity) of primary (observing) satellite. Units: [m; m/s]

  • xt (np.ndarray) – Inertial state (position and velocity) of the target satellite. Units: [m; m/s]

Returns

Rotation matrix transforming from the Cartesian frame to the RTN frame.

Return type

np.ndarray

  • xt (np.ndarray) – Inertial state (position and velocity) of the target satellite. Units: [m; m/s]

Returns

np.ndarray Rotation matrix transforming from

the RTN frame to the Cartesian frame.

Compute the radial, along-track, cross-track (RTN) coordinates of a target satellite in the primary satellite’s RTN frame.

The RTN frame is also commonly refered to as the local-vertical, local-horizontal (LVLH) frame.

  • xt (np.ndarray) – Inertial state (position and velocity) of the target satellite. Units: [m; m/s]

Returns

Position and velocity of the target relative of the

observing satellite in the RTN. Units: [m; m/s]

Return type

np.ndarray
