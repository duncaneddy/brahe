# Walker Constellation Generator Builder

Builder for `WalkerConstellationGenerator`. `builder()` takes the six required inputs (`t`, `p`, `f`, `semi_major_axis`, `inclination`, `epoch`) directly as arguments. Optional inputs are set through chained setters and default to the same values as the flat constructor's optional parameters.

For understanding Walker constellation geometry and usage examples, see [Walker Constellations](../../learn/orbits/walker.md).
