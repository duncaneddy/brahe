# FileEOPProvider

**Module**: `brahe.eop`

Load Earth Orientation Parameters from IERS data files, in either Standard
(`finals2000A.all`) or C04 (`eopc04_IAU2000.XX`) format.

::: brahe.FileEOPProvider
    options:
      show_root_heading: true
      show_root_full_path: false
