# Plots

**Module**: `brahe.plots`

Visualization tools for astrodynamics data with support for matplotlib and plotly backends.

## Sub-modules

- **[Ground Track Plotting](ground_tracks.md)** - Ground track visualization with communication cones and polygon zones
- **[Access Geometry Plotting](access_geometry.md)** - Polar plots and elevation profiles for access windows
- **[Orbital Trajectories](orbital_trajectories.md)** - Time-series plots of orbital elements (Cartesian and Keplerian)
- **[3D Trajectory](3d_trajectory.md)** - 3D visualization of satellite trajectories in ECI frame
- **[Gabbard Diagram](gabbard.md)** - Orbital period vs apogee/perigee altitude scatter plots
- **[Estimation State Plots](estimation_state.md)** - Filter error and state value plots with covariance bands
- **[Measurement Residual Plots](estimation_residuals.md)** - Pre-fit/post-fit residual diagnostics
- **[Marginal Distribution Plots](estimation_marginal.md)** - 2D covariance ellipses with marginal densities
