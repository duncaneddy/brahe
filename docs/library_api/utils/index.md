# Utilities API Reference

This section provides detailed API documentation for all utility functions in Brahe.

## Overview

The utilities module provides support functions for:

- **[Caching](caching.md)**: Cache directory management for downloaded data
- **[Multithreading](threading.md)**: Thread pool configuration for parallel computation
- **[String Formatting](formatting.md)**: Human-readable output formatting
